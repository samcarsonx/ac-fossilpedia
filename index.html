About Über Acerca de À propos de

This is a small open-source web applet I made to help keep track of the fossils you have and don't have in Animal Crossing: New Horizons. It uses jQuery and LocalStorage to handle the JavaScript and saving of data (remembering what checkboxes you've clicked).

Dies ist ein kleines Open-Source-Web-Applet, das ich erstellt habe, um den Überblick über die Fossilien zu behalten, die man in Animal Crossing: New Horizons hat und nicht hat. Es verwendet jQuery und LocalStorage, um das JavaScript zu verwalten und Daten zu speichern (wobei du dir merken musst, auf welche Kontrollkästchen du geklickt hast).

Este es un pequeño applet web de código abierto que hice para ayudar a hacer un seguimiento de los fósiles que tienes y no tienes en Animal Crossing: New Horizons. Usa jQuery y LocalStorage para manejar el JavaScript y guardar datos (recordando en qué casillas de verificación has hecho clic).

Il s'agit d'une petite applet Web open source que j'ai créée pour aider à garder une trace des fossiles que vous avez et que vous n'avez pas dans Animal Crossing: New Horizons. Il utilise jQuery et LocalStorage pour gérer JavaScript et l'enregistrement des données (en se souvenant des cases à cocher sur lesquelles vous avez cliqué).
